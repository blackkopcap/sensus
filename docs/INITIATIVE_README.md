# Initiative System - Helen AI 💭

## Обновления системы инициативного общения

### ✅ Исправлено

1. **Снижен порог отправки**: priority > 0.4 + >1ч молчания → отправлять
2. **Увеличен cooldown**: 15 мин → 2 часа между инициативами  
3. **Добавлен дневной лимит**: максимум 3 инициативы в день
4. **Конкретные временные триггеры**:
   - Утреннее приветствие (7:00-9:00) после >6ч молчания
   - Вечерний чекин (18:00-20:00) после >5ч молчания
   - "Давно не общались" после >3ч молчания
5. **Упрощена логика heartbeat**: прямой возврат текста без записи в базу

### 🎯 Новые правила работы

**Приоритеты отправки:**
- `priority > 0.7` — отправляется сразу (high priority)
- `priority > 0.4` — отправляется после >1ч молчания  
- `priority ≤ 0.4` — игнорируется

**Временные ограничения:**
- Cooldown: 2 часа между инициативами
- Quiet hours: 23:00-07:00 (не писать)
- Лимит: 3 инициативы в день
- Мысли истекают через 24 часа

**Конкретные триггеры (генерируются автоматически):**
- Утром (7-9): "Доброе утро! ☀️ Как дела, готов к новому дню?"
- Вечером (18-20): "Как прошёл день? Всё получилось как планировал?"
- После >3ч: "Давно не общались! Как дела? 😊"

### 🔧 Команды

```bash
# Статус системы
node scripts/initiative.js status

# Добавить мысль в очередь
node scripts/initiative.js queue --thought "Текст мысли" --priority 0.6 --category "idea"

# Проверить решение о том, говорить ли сейчас
node scripts/initiative.js check

# Heartbeat интеграция (используется автоматически)
node scripts/initiative.js heartbeat

# Очистить очередь
node scripts/initiative.js clear --all
```
